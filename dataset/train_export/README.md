Place the contents of ArcGIS Pro's `Export Training Data For Deep Learning` output in this folder.

Important:

- Copy the `images` folder into this folder.
- Copy the `labels` folder into this folder.
- Copy the utility files that ArcGIS exported alongside them, such as `map.txt`, `stats.json`, or `.emd` files.
- Do not copy the outer export folder itself as an extra nested folder.

After copying, this folder should look roughly like:

```text
dataset/train_export/
  images/
  labels/
  map.txt
  stats.json
```
